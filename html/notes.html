‚úÖ Section 1: HTML Basics

üîπ What is HTML?

HTML (HyperText Markup Language) is the standard language used to structure web pages. It uses tags to describe content.

üîπ Basic HTML Document Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
Tag Description Example Why & When to Use
<!DOCTYPE html> Declares HTML5 version <!DOCTYPE html> Must be first line; tells browser the HTML version.
<html> Root element <html>...</html> Wraps all page content.
<head> Metadata, links, scripts <head><title>Title</title></head> SEO, CSS links, scripts go here.
<body> Visible page content <body><h1>Heading</h1></body> Everything user sees lives here.

‚úÖ Section 2: Text & Link Elements

Tag Description Example Why & When to Use
<h1>-<h6> Headings (h1 is largest) <h1>Title</h1> Use for proper document structure and SEO.
<p> Paragraph <p>This is a paragraph.</p> For regular text content.
<a> Anchor/hyperlink <a href="https://google.com">Google</a> For navigation, linking to pages.
<br> Line break Line1<br>Line2 Avoid for spacing‚Äîuse only for poems, addresses.
<strong> Strong/bold text <strong>Important</strong> Semantically important bold text.
<em> Emphasized/italic text <em>Note</em> Semantically emphasized text.

‚úÖ Section 3: Lists

Tag Description Example Why & When to Use
<ul> Unordered list <ul><li>Item</li></ul> Bullet points, unordered info.
<ol> Ordered list <ol><li>Step</li></ol> Step-by-step, numbered tasks.
<li> List item <li>Point</li> Used inside ul, ol.
<dl> Description list <dl><dt>HTML</dt><dd>Markup</dd></dl> Glossary, FAQs.

‚úÖ Section 4: Images & Media

Tag Description Example Why & When to Use
<img> Embeds image <img src="cat.jpg" alt="cat"> Add visuals, branding. Always use alt.
<video> Embeds video <video controls><source src="video.mp4"></video> Tutorials, media-rich sites.
<audio> Embeds audio <audio controls><source src="song.mp3"></audio> Background music, podcasts.

‚úÖ Section 5: Forms & Inputs

Tag Description Example Why & When to Use
<form> Form wrapper <form action="submit.php">...</form> For collecting user input.
<input> Various fields <input type="text"> Text, checkbox, radio, etc.
<textarea> Multi-line input <textarea></textarea> Long text like messages.
<button> Button element <button>Submit</button> Submit forms, click actions.
<label> Field label <label for="name">Name:</label> Improves accessibility and UX.

‚úÖ Section 6: Semantic HTML

Tag Description Example Why & When to Use
<header> Page header <header>...</header> Intro content, logo, nav bar.
<nav> Navigation links <nav><a>...</a></nav> Site menus, internal links.
<section> Page sections <section>...</section> Group content by topic.
<article> Standalone content <article>Blog</article> News, blog posts.
<aside> Side info <aside>Tip</aside> Ads, side notes.
<footer> Bottom section <footer>Copyright</footer> End of page info.

üî∏ Why use semantic HTML?

  • Improves SEO
  • Easier to read code
  • Screen readers can navigate properly

‚úÖ Section 7: CSS Overview

üîπ What is CSS?

CSS (Cascading Style Sheets) controls how HTML elements look: colors, layout, size, fonts, and more.

üîπ Ways to Apply CSS:

Method Syntax Use When
Inline <p style="color:red;"> Small changes or testing only.
Internal <style>...</style> Small single-file projects.
External <link rel="stylesheet"> Best for large sites, reusable styles.

‚úÖ Section 8: Common CSS Properties

‚úÖ Section 9: Box Model Explained

Each HTML element is a box with layers:

[ margin ]   ‚Üê space outside the element
[ border ]   ‚Üê boundary of the box
[ padding ]  ‚Üê space inside border
[ content ]  ‚Üê actual text or image

üîπ Understand this to fix spacing/layout issues.

‚úÖ Section 10: Layout with Display, Flexbox & Grid

üîπ Display Types

Value What It Does
block Fills full width (div, p)
inline Stays inline (span, a)
flex Flexible box layout
grid Two-dimensional layout

üîπ Flexbox Example:

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

üìå Use for horizontal/vertical layout control easily.

üîπ Grid Example:

.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

üìå Use grid for column/row placement and structure.
